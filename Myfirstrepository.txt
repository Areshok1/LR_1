Контрольні питання:
1)Git — розподілена система керування версіями файлів та спільної роботи.
2)Git має три основних стани, в яких можуть перебувати ваші файли: збережений у коміті , змінений та індексований.
3)Команда git add приймає шлях файлу або директорії. Якщо це директорія, команда додає усі файли в цій директорії рекурсивно.
4)Перевірка статусу ваших файлів
5)Коли ви виходите з редактору, Git створює коміт з цим повідомленням коміту.
6)Ініціалізація Git у папці проекту.
7)коротку інформацію про те, що змінилося в репозиторії після цього коміту 
8)git log
9)Команда git checkout дозволяє переміщатися між гілками, створеними командою git branch.
